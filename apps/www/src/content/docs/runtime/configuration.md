Configuration should live near the application boundary. Keep runtime defaults explicit so production behavior is easy to review.

## Model defaults

Set the model, temperature, and output mode where the runtime is created.

## Capability flags

Enable tools, retrieval, tracing, and streaming only when a product surface needs them.
