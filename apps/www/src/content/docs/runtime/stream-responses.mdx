Streaming keeps the interface responsive while the model is still working. Anvia streams structured parts instead of forcing every client to parse raw text.

## Use server events

Expose runtime output as server-sent events when you need a browser-friendly transport.

```ts
const stream = await runtime.stream({
  input: "Draft a reply to this customer.",
});

return stream.toResponse();
```

## Handle typed parts

Clients can render text, tool calls, tool results, and final metadata independently.

```ts
for await (const part of stream) {
  if (part.type === "text") {
    appendMessage(part.text);
  }
}
```

Typed stream parts make it easier to build polished agent interfaces without coupling UI code to one provider format.
